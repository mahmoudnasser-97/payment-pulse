# PaymentPulse

A real-time payment transaction analytics and fraud detection platform built with Kafka, Spark, Delta Lake, Dagster, and Streamlit.

## Stack
- Apache Kafka — real-time event streaming
- Apache Spark — stream processing and ML scoring
- MinIO — S3-compatible data lake (Bronze / Silver / Gold)
- PostgreSQL — data warehouse
- Dagster — pipeline orchestration
- Streamlit — live dashboard
- Redis — real-time feature store
